## Method 1: Browser Developer Tools (Recommended)

### Using Chrome/Edge DevTools:

1. **Open DevTools**
   - Press `F12` or `Ctrl+Shift+I` (Windows/Linux)
   - Press `Cmd+Option+I` (Mac)

2. **Toggle Device Toolbar**
   - Press `Ctrl+Shift+M` (Windows/Linux)
   - Press `Cmd+Shift+M` (Mac)
   - Or click the mobile/tablet icon in DevTools

3. **Select Mobile Device**
   - Choose "iPhone 12 Pro" or "iPhone 13 Pro" (390 x 844)
   - Or custom dimensions: 390 x 844 pixels

4. **Capture Screenshot**
   - Click the three-dot menu (‚ãÆ) in the Device Toolbar
   - Select "Capture screenshot"
   - Image will download automatically

5. **For Full-Page Screenshots**
   - Open Command Menu: `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
   - Type "screenshot"
   - Select "Capture full size screenshot"

## Screenshot Specifications for Presentation

### For PowerPoint/Keynote Presentations:

**Recommended Dimensions:**
- **Mobile**: 390 x 844 pixels (iPhone 13 Pro ratio)
- **Export Format**: PNG (for transparency) or JPG (smaller file size)
- **Resolution**: 72 DPI for screens, 300 DPI for print

### Organization Tips:

1. **Create Folders by Section:**
