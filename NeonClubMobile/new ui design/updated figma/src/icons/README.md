## Usage

Each SVG file can be:
1. Imported directly into React components
2. Used in image tags
3. Converted to PNG/other formats using design tools
4. Embedded directly in HTML

## Color Customization

All SVGs use `currentColor` for the stroke/fill, so they inherit the text color of their parent element. You can customize by:
- Setting CSS `color` property
- Using Tailwind classes like `text-purple-500`
- Directly editing the SVG fill/stroke attributes
